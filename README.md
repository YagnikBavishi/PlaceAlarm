About The Appliction
============
Location Based Remainder Using GPS Service is an application to set an alarm for  android  mobile phones, this service is based on the location of device and user can also see the places which are near to him/her using current location of him/her. Using this application, when the user comes into particular area (destination) then the alarm will ring. Also, User can set to send message/email when he/she near to destination, So that the person who is coming to peak him/her can get notification that him/her is now only
__km away from the destination.
